#+title: Kernel trick


The kernel trick is the idea of choosing a positive definite kernel so
that the data is then embedded in a [[id:2c88b6ee-ba2b-42ab-a830-7199d018d7c8][RKHS]].
Linear algorithm can then be used directly in a non-linear case.
